# Как менять контент сайта DamuMarket

**99% правок делается в одном файле — `data.js`.** Открывайте его в любом текстовом редакторе (даже в «Блокноте»), правьте текст между кавычек, сохраняйте — и перезагружайте страницу в браузере. Готово.

> Внимание: НЕ удаляйте кавычки `'` и запятые `,` в конце строк. Если случайно удалили — страница может перестать загружаться. В этом случае откройте `data.js` в браузере (F12 → Console) и посмотрите ошибку.

---

## 1. Поменять телефон / WhatsApp

В файле `data.js`, секция `contacts`:

```js
phone_display: '+7 700 215 28 93',   // как показывается на сайте
phone_tel:     '+77002152893',       // как набирается (для tel:)
wa_number:     '77002152893',        // для wa.me/<число>
```

Если телефон поменялся целиком — поменяйте **во всех трёх местах**.

---

## 2. Адрес и часы работы

Та же секция `contacts`:

```js
address: 'г. Алматы, ул. Толе би 285',
hours_lines: [
  { day: 'Пн–Сб', time: '9:00 – 19:00' },
  { day: 'Воскресенье', time: '10:00 – 17:00' }
],
```

Адрес ещё дополнительно прописан в `index.html` (для поисковиков, в JSON-LD блоке и в подвале) — если переезд, дайте знать программисту, нужно поменять там тоже.

---

## 3. Реквизиты для безнала

```js
bin:  '',     // ваш БИН — заполните строкой '123456789012'
iban: '',     // IBAN, например 'KZ12345678901234567890'
bank: '',     // например 'АО Halyk Bank'
```

Когда заполните — на сайте автоматически появятся реальные данные вместо «уточняется».

---

## 4. Поменять цену / название / наличие товара

Найдите нужную категорию (например `id: 'mixes'`), затем нужную позицию в `items`:

```js
{ name: 'Штукатурка Knauf Ротбанд 30 кг', brand: 'Knauf', price_from: 5500, available: true },
```

- `name` — название
- `brand` — бренд (или `'—'` если без бренда)
- `price_from` — цена «от», только число, без пробелов и «₸»
- `available: true` → в наличии. `available: false` → под заказ.

---

## 5. Добавить новую позицию в категорию

Скопируйте любую строчку, вставьте под ней, поменяйте текст. **Не забудьте запятую в конце предыдущей строки.**

```js
items: [
  { name: 'Старая позиция', brand: 'X', price_from: 1000, available: true },
  { name: 'Новая позиция',  brand: 'Y', price_from: 2000, available: true }   // <- новая
]
```

---

## 6. Добавить новую категорию

В массиве `categories: [...]` скопируйте целиком одну категорию (например `mixes`), вставьте ниже, поменяйте:

- `id` — латиницей, без пробелов (`'wallpaper'`)
- `title` — название («Обои»)
- `description` — короткое описание
- `image` — ссылка на картинку (см. п. 7)
- `items` — список позиций

---

## 7. Поменять картинку категории или героя

Картинки берутся с Unsplash. Чтобы поменять:

1. Зайдите на https://unsplash.com/s/photos/paint (или другой запрос)
2. Откройте нужную фотку
3. Нажмите «Download», в URL появится номер фото вида `photo-1572981779307-38b8cabb2407`
4. Подставьте в `data.js`:

   ```js
   image: 'https://images.unsplash.com/photo-1572981779307-38b8cabb2407?w=600&auto=format&fit=crop&q=70'
   ```

Параметр `?w=600&auto=format&fit=crop&q=70` оставьте — он гарантирует, что картинка лёгкая (~50 КБ) и красивая.

Для hero (первый экран) можно `w=800` и `q=75`.

---

## 8. Отзывы

В массиве `reviews: [...]`. Каждый отзыв — карточка:

```js
{
  name: 'Айдар',
  role: 'Прораб, ремонт квартиры',
  text: 'Заказывал гипсокартон...',
  date: 'март 2026'
}
```

Лучше три отзыва — реальных, с разной историей (частник / прораб / организация). Звёздочки/рейтинги не добавляем — фейково смотрятся.

---

## 9. Часто задаваемые вопросы (FAQ)

Массив `faq: [...]`. `q` — вопрос, `a` — ответ. Сократите до 4–5 самых горячих, иначе никто не прочитает.

---

## 10. Что-то совсем сломалось

1. Откройте сайт в Chrome / Safari
2. Нажмите F12 (или ⌥⌘I) → вкладка Console
3. Если красный текст с ошибкой — скопируйте, отправьте программисту
4. Часто причина: пропущена запятая или кавычка в `data.js`

- [ ] Реальный телефон в `phone_display`, `phone_tel`, `wa_number`
- [ ] Реальный адрес магазина в `address`
- [ ] Реальные часы работы в `hours_lines`
- [ ] Реквизиты (БИН/IBAN/банк) или указано, что у менеджера
- [ ] 2–3 живых отзыва вместо моих плейсхолдеров
- [ ] Цены актуальные (хотя бы на топ-5 позиций в каждой категории)
- [ ] Открыли с телефона, пощёлкали кнопки WhatsApp — текст в сообщении нормальный
